# Derm AI Nexus - Quick Run

Run everything from one terminal with one command:

```bash
npm run dev
```

You should see:

`Derm AI Nexus is running at http://localhost:3000`

Then open that address in your browser.

## First-time setup

```bash
npm run install:all
```

## Environment

Create `frontend/.env.local`:

```bash
OPENAI_API_KEY=your_openai_api_key_here
```
